# 坦克大战 - Battle City

经典坦克大战（Battle City）的 HTML5 网页版实现。纯前端单页面游戏，打开即玩，无需安装。

## 在线试玩

直接浏览器打开 `index.html`，或通过 GitHub Pages 访问。

## 游戏操作

| 操作 | 按键 |
|------|------|
| 移动 | `↑` `↓` `←` `→` 或 `W` `A` `S` `D` |
| 开火 | `空格` 或 `J` |
| 开始 | 点击「开始游戏」按钮 |

触屏设备会自动显示虚拟方向键和开火按钮。

## 游戏规则

- 控制**黄色坦克**，消灭所有红色敌军坦克
- 保护屏幕底部的**基地（鹰旗）**不被摧毁
- 砖墙可被子弹击毁，钢墙不可破坏
- 每波 10 个敌人，逐波增加难度（速度提升、射速加快、部分敌人拥有双倍血量）
- 玩家有 **3 条生命**，全部用尽则游戏结束

## 技术特性

- 纯 HTML + CSS + JavaScript，单文件无依赖
- Canvas 2D 渲染
- 敌人 AI（追踪玩家、随机移动、自主射击）
- 粒子爆炸特效
- Web Audio API 音效
- 响应式布局，支持桌面和移动端
- 触摸设备虚拟按键
